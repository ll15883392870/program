#### 项目使用

**1.音乐下载器GUI界面**

在终端运行如下命令即可:

```python
python musicdlgui.py
```

**2.歌手歌词分析**

在终端运行如下命令即可:

```python
python singerlyricsanalysis.py
```

**3.歌词获取歌曲片段**

在终端运行如下命令即可:

```python
python searchlyrics.py
```
